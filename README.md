## ✨ ویژگی ها

- 🤖 **هوش مصنوعی قدرتمند** - اتصال به Groq API با مدل LLaMA 3.3 70B
- 🎨 **رابط کاربری شبیه ChatGPT** - با طراحی مذهبی-مسجدی
- 💾 **ذخیره تاریخچه گفتگوها** - در مرورگر کاربر (LocalStorage)
- ⚙️ **پنل مدیریت کامل** - با دسترسی از طریق رمز عبور
- 📱 **ریسپانسیو** - سازگار با موبایل و دسکتاپ

## 🚀 نصب و راه اندازی

### روش 1: اجرای مستقیم

1. فایل های پروژه را دانلود کنید
2. فایل `index.html` را در مرورگر باز کنید

### روش 2: راه اندازی روی سرور

```bash
# Clone کردن پروژه
git clone https://github.com/username/oay-yaqin.git
cd oay-yaqin

# اجرا با Python
python -m http.server 8000

# یا با Node.js
npx serve .
```

سپس مرورگر را به آدرس `http://localhost:8000` باز کنید.
